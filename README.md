**`Desenvolvedor de Software Multiplataforma`**

Me chamo Bruno José dos Santos Silva, tenho 25 anos e sou apaixonado por tecnologia e desenvolvimento de software. Atualmente, estou cursando **Desenvolvimento de Software Multiplataforma**. Estou sempre buscando aprender novas tecnologias e criar soluções úteis e bem estruturadas.
